Two Sum Solutions

Brute Force Approach

To use the brute force approach, we simply loop through each element x and find if there is another value that equals to the target - x.

Example in python3:

class Solution:

    def twoSum(self, nums, target):

        """

        :type nums: List[int]

        :type target: int

        :rtype: List[int]

        """

        seen = {}

        for index, num in enumerate(nums):

            other = target - num

            if other in seen:

                return [seen[other], index]

            else:

                seen[num] = index

        return []
